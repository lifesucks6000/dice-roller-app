# Dice Roller

Dice roller is a Flutter-based mobile application designed simply to do one task, rolling a dice.

## Description

This app give you:
- Random number between 1-6 when you roll the dice.

## Getting Started

### Prerequisites

- Flutter SDK (version 2.0 or later)
- Dart (version 2.12 or later)
- Android Studio or VS Code with Flutter extensions

### Installation

1. Clone the repository:

```bash
git clone https://github.com/lifesucks6000/dice-roller-app.git
```

2. Navigate to the project directory:

```bash
cd dice-roller-app
```

3. Get the dependencies:

```bash
flutter pub get
```

### Running the app

To run the app on a connected device or emulator:

```bash
flutter run
```
